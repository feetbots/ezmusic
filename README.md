# EzMusic v1.0.0
A MacOS spotify clone for youtube videos.

## Requirements
- MacOS Lion (10.7+)
- Requirements.txt
- Xcode CLT
```
pip(3) install -r requirements.txt
xcode-select --install
```

## Installation & Usage
Download and run the installer file to get started.\
\
You can launch the app by typing `ezmusic` into a terminal.
